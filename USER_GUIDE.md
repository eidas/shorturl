# USER GUIDE

## 詳細な使用ガイド
このガイドでは、ShortURLサービスの使用方法について詳しく説明します。

## URL追加のステップバイステップ手順
1. サービスを起動します。
2. Webインターフェースにアクセスします。
3. フォームに元のURLを入力します。
4. 必要に応じてカスタムエイリアスを設定します。
5. 「短縮URLを生成」ボタンをクリックします。

## トラブルシューティング
- **短縮URLが機能しない場合**: 入力したURLが正しいか確認してください。
- **カスタムエイリアスが使用できない場合**: 既に使用されている可能性があります。別のエイリアスを試してください。

## ベストプラクティス
- URLを短縮する前に、元のURLが正しいか確認してください。
- カスタムエイリアスを設定する場合は、簡潔で覚えやすいものを選んでください。

## よくある質問（FAQ）

### Q: 短縮URLの有効期限はありますか？
A: デフォルトでは有効期限はありませんが、設定で変更可能です。

### Q: 短縮URLのクリック数を確認できますか？
A: はい、管理画面で確認できます。

### Q: カスタムドメインを使用できますか？
A: はい、設定でカスタムドメインを追加できます。
