# ShortURL

## プロジェクト概要
ShortURLは、短縮URLを生成し、管理するためのサービスです。このプロジェクトは、簡単に使用できるインターフェースと柔軟な設定オプションを提供します。

### 主な機能
- 短縮URLの生成
- URLのクリック数の分析
- カスタムエイリアスの設定
- URLの有効期限の設定

## インストール・セットアップ

### 前提条件
- Node.js (バージョン16以上)
- npm (バージョン7以上)

### クローン手順
```bash
git clone https://github.com/eidas/shorturl.git
cd shorturl
```

### 依存関係インストール
```bash
npm install
```

## 使用方法

### URLの追加方法
1. サービスを起動します。
2. WebインターフェースまたはAPIを使用してURLを追加します。

### ローカルビルド方法
```bash
npm run build
```

### デプロイ方法
```bash
npm run deploy
```

## 設定ファイルの説明

### urls.jsonの構造
URLのリストとその設定を管理します。

### site.jsonの構造
サイト全体の設定を管理します。

## ビルドコマンド
- `npm run build`: プロジェクトをビルドします。
- `npm run validate`: 設定ファイルを検証します。
- `npm test`: テストを実行します。

## GitHub Pagesの設定手順
1. リポジトリのSettingsに移動します。
2. Pagesセクションでブランチを選択します。

## カスタムドメイン設定（オプション）
1. DNS設定でCNAMEレコードを追加します。
2. GitHub Pagesの設定でカスタムドメインを指定します。
